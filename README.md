## Проєкт: Алгоритм Беллмана-Форда

### Призначення
Експериментальне дослідження часової складності алгоритму Беллмана-Форда на зважених орієнтованих графах. Проведено порівняльний аналіз продуктивності двох структур даних:
1.  Списки суміжності (Adjacency List).
2.  Матриця суміжності (Adjacency Matrix).

### Структура проєкту
| Файл | Опис |
| :--- | :--- |
| `graph_lib.py` | Містить клас `Graph`, функцію генерації графа та дві реалізації алгоритму Беллмана-Форда. |
| `experiments.py` | Основний скрипт для запуску серії тестів, збору даних і збереження їх у CSV. |
| `draw_plots.py` | Скрипт для автоматичної обробки даних і побудови фінальних графіків. |
| `results_extended.csv` | Файл із "сирими" результатами вимірювань часу виконання. |

### Запуск експериментів
Для відтворення результатів необхідно:
1.  Клонувати репозиторій.
2.  Встановити необхідні бібліотеки: `pip install pandas matplotlib`.
3.  Запустити скрипт експериментів:
    ```bash
    python experiments.py
    ```
4.  Графіки будуть згенеровані автоматично.
